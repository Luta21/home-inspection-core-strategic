Final Design — Service Page Layout

Complete section flow for all 6 service pages. Each page is an immersive long-scroll conversion funnel with 10 sections.

Service Page Blueprint
10-Section Conversion Funnel
01
Full-Width Cinematic Hero
Centered dramatic headline + gold accent tagline. Dual CTAs (Programeaza + Telefon). Trust stats strip at bottom (1000+ proprietati, 15+ ani, 48h raport). Scroll indicator. GSAP staggered text reveal.
02
Informative Concern Section
Professional advisor tone — "De ce este importanta verificarea?" Educational approach with data points. Positions Core Strategic as the knowledgeable partner. No fear-mongering, just informed guidance. GSAP fade-in reveal.
03
Solution Overview
What this specific service does and why it matters. Service-specific value proposition. Key benefits in a concise grid or feature list. Gold accent highlights on key phrases.
04
Horizontal Stepper — Methodology
Interactive numbered circles connected by gold line. Each step activates on scroll — gold line fills progressively. Content panel expands below active step. 4-6 steps per service. GSAP ScrollTrigger scrub animation.
05
Equipment — Horizontal Scroll Cards
Equal-weight tool cards in horizontal scroll. Each card: Lucide icon (NO emojis), tool name, model, one-line specs. Peek of next card for scroll affordance. Service-specific equipment highlighted. GSAP stagger entrance.
06
Deliverables Checklist
Vertical list with gold checkmarks. Every item the client receives: raport PDF detaliat, documentatie foto, estimare costuri remediere, recomandari tehnice, consultanta post-inspectie. Makes value tangible.
07
CTA Banner — Mid-Page
Reuses existing CTABanner component. Service-specific headline. Dual action: phone call + schedule inspection. Catches visitors who are already convinced and don't need to scroll further.
08
Related Services — Card Row
3 compact cards matching homepage ServicesSection style. Excludes current service. Lucide icon, title, short desc, arrow CTA. GSAP stagger + parallax tilt. Strong internal linking for SEO.
09
Service-Specific FAQ
Accordion with 4-6 questions unique to this service. Addresses objections and builds confidence. Gold expand/collapse icons. SEO-rich content — targets long-tail keywords. Reuses FAQSection pattern.
10
Final CTA Section
Full-width closing conversion block. Phone number, WhatsApp, schedule form link. Service-specific closing headline. Reuses CTASection pattern from homepage.
Design Principles
✓ Dark luxury aesthetic (black + gold)
✓ Playfair Display + Inter typography
✓ GSAP ScrollTrigger animations throughout
✓ Lucide icons only — zero emojis
✓ prefers-reduced-motion respected
✓ Data-driven from constants (6 services)
✓ SEO: unique metadata + JSON-LD per page
✓ Lenis smooth scroll integration
